Projects

Guess The Word Game

Guess a word by typing one letter at a time. Guess the wrong letter, you lose a turn! Get all the letters before your turns are up, and win! Through the use of JSX and manipulating the DOM, you get to practice your letter-guessing skills!

Hotel Booking App

Hotel app that allows you to search and filter for the hotel room you are looking for. With the use of JSX, managing the data flow and manipulating the DOM, you are able to narrow down your hotel selection with the filters available.

Name Tag Generator

Name tag generator that lets you create multiple name tags. By managing the data flow and using Code Sandbox and JSX, you can add, personalize and delete name tags with ease.

Sticky Notes

Fun app that lets users create sticky notes! With the use of React and Code SandBox, you can add and delete sticky notes, edit as needed and search for specific notes.

Unplugged Retreat

Multi-page responsive website. Through the use of responsive typography and media queries, you can view the page on mobile, tablet and desktop screens with ease.
